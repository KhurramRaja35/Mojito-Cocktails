## Description

Mojito Cocktails is a **stunning, interactive landing page** built using **React**, **Tailwind CSS**, and **GSAP**. The project focuses on creating an immersive user experience with **scroll-based animations**, **timeline sequences**, and **animated video content**, all seamlessly integrated into a responsive modern design.

Developed in **WebStorm IDE** with a **Vite** setup for blazing-fast development and hot module replacement.

---

## Features

- **React & Tailwind CSS:** Clean component-based architecture with modern styling.
- **GSAP Animations:**  
  - Scroll-triggered animations for smooth, engaging transitions  
  - Timeline animations to synchronize multiple elements  
  - Video animation triggered by scrolling
- **Responsive Design:** Mobile, tablet, and desktop ready.
- **Interactive Components:** Hover effects, sliders, and dynamic cocktail content.
- **Modern Development Setup:** Built with Vite for fast compilation and optimized performance.

## Installation

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/mojito-cocktails.git
cd mojito-cocktails
```

2. **Install dependencies:**
```bash
npm install
```

3. **Run the development server:**
```bash
npm run dev
```

4. **Open in browser:**
```bash
Visit http://localhost:5173 (Vite default port)
```

## Usage
- Scroll through the landing page to see smooth GSAP-driven animations.

- Hover and interact with cocktail cards and buttons.

- Watch the video animate as you scroll down the page.

- Explore the beautifully crafted Mojito-themed UI.
